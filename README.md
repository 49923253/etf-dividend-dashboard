# ETF 配息儀表板（MVP）

這是一個純前端的靜態頁面（無框架、免安裝），用來展示「台股 ETF 股價 + 即將除息/評價/發放日」並可排序/篩選，並能標註「季配息」與「科技股」等欄位。

> 目前附的是 `data.sample.json` 範例資料（假資料格式）。
> 你要接真資料時，我可以再加一個抓取/更新腳本（或改成讀你指定來源匯出的 CSV/JSON）。

## 功能

- 依「除息日」由近到遠排序（可切換排序欄位）
- 篩選：
  - 只看「即將除息」（可設定 N 天內）
  - 只看「季配息」
  - 只看「科技股」
- 清楚欄位：
  - ETF 代號 / 名稱
  - 目前股價
  - 除息日、收益分配評價日、發放日
  - 配息頻率（季/月/半年/年/不明）
  - 標籤：季配息、科技股

## 使用方式

1. 直接用瀏覽器打開：`index.html`
2. 若你想避免瀏覽器 CORS/檔案限制，建議用本機小伺服器：
   - PowerShell：
     - `cd etf-dividend-dashboard`
     - `python -m http.server 8000`
   - 然後開： http://127.0.0.1:8000/
